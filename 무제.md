### 문제

N개의 정수를 입력받아 **입력된 순서의 반대로** 출력하라.

```
입력:
5
1 2 3 4 5

출력:
5 4 3 2 1
```

**조건:** 배열 인덱스를 역순으로 접근하는 방식(`arr[n-1]`부터) 말고, **스택의 push/pop 구조를 직접 구현해서** 풀 것.

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
	int N;
	scanf("%d", &N);
	
	int* stack = (int*)malloc(sizeof(int) * N);
	int top = -1;
	
	for (int i=0; i<N; i++) {
		int val;
		scanf("%d", &val);
		stack[++top] = val;
	}
	
	for (int i=0; i<N; i++) {
		printf("%d ", stack[top--]);
	}
	
	free(stack);
	stack = NULL;
	
	return 0;
}
```

### 문제

`(` 와 `)` 로만 이루어진 문자열이 주어졌을 때, **올바른 괄호 문자열이면 1, 아니면 0**을 출력하라.

```
입력: (())()
출력: 1

입력: ())
출력: 0

입력: ((()
출력: 0
```

**올바른 괄호 조건**

- 모든 `(` 는 반드시 대응하는 `)` 가 있어야 함
- `)` 가 나왔을 때 대응할 `(` 가 없으면 안 됨

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main() {
	char* str = (char*)malloc(sizeof(char)*1000);
	scanf("%s", str);
	
	int len = strlen(str);
	
	char* stack = (char*)malloc(sizeof(char)*1000);
	int top = -1;
	
	for (int i=0; i<len; i++) {
		if (str[i] == '(') stack[++top] = '(';
		else if (str[i] == ')') {
			if (top == -1) {
				printf("0");
				return 0;
			}
			else top--;
		}
	}
	
	if (top == -1) printf("1");
	else printf("0");
	
	free(str);
	free(stack);
	
	str = NULL;
	stack = NULL;
	
	return 0;
}
```
