### 문제

N개의 정수를 입력받아 **입력된 순서의 반대로** 출력하라.

```
입력:
5
1 2 3 4 5

출력:
5 4 3 2 1
```

**조건:** 배열 인덱스를 역순으로 접근하는 방식(`arr[n-1]`부터) 말고, **스택의 push/pop 구조를 직접 구현해서** 풀 것.

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
	int N;
	scanf("%d", &N);
	
	int* stack = (int*)malloc(sizeof(int) * N);
	int top = -1;
	
	for (int i=0; i<N; i++) {
		int val;
		scanf("%d", &val);
		stack[++top] = val;
	}
	
	for (int i=0; i<N; i++) {
		printf("%d ", stack[top--]);
	}
	
	free(stack);
	stack = NULL;
	
	return 0;
}
```
