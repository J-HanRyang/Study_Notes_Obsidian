# 수 정렬하기

## 문제 요약
N개의 정수를 오름차순으로 정렬해서 출력 (N ≤ 1,000,000)

## 풀이 접근
N이 크면 스택 오버플로우 → malloc으로 힙 할당 후 qsort

## 코드
```c
#include <stdio.h>
#include <stdlib.h>

int compare(const void* a, const void* b) {
    return *(int*)a - *(int*)b;
}

int main() {
    int N;
    scanf("%d", &N);

    int *arr = (int*)malloc(sizeof(int) * N);
    for (int i = 0; i < N; i++) {
        scanf("%d", &arr[i]);
    }

    qsort(arr, N, sizeof(int), compare);

    for (int i = 0; i < N; i++) {
        printf("%d\n", arr[i]);
    }

    free(arr);
    arr = NULL;

    return 0;
}
```

## 핵심 포인트
- N이 크면 스택 선언 대신 malloc 사용
