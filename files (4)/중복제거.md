# 중복 제거

## 문제 요약
N개의 정수에서 중복을 제거하고 오름차순 출력

## 풀이 접근
정렬 후 앞 원소와 비교 → 추가 배열 없이 출력만으로 해결

## 코드
```c
#include <stdio.h>
#include <stdlib.h>

int compare(const void* a, const void* b) {
    return *(int*)a - *(int*)b;
}

int main() {
    int N;
    scanf("%d", &N);

    int *arr = (int*)malloc(sizeof(int) * N);
    for (int i = 0; i < N; i++) {
        scanf("%d", &arr[i]);
    }

    qsort(arr, N, sizeof(int), compare);

    printf("%d ", arr[0]);
    for (int i = 1; i < N; i++) {
        if (arr[i] != arr[i-1]) printf("%d ", arr[i]);
    }

    free(arr);
    arr = NULL;

    return 0;
}
```

## 핵심 포인트
- 정렬 후 인접 비교로 중복 제거
- 추가 배열 불필요
