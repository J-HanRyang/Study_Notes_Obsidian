# 투 포인터 응용 (부분합)

## 한줄 요약
> 연속된 구간의 합이 T인 경우 찾기 — O(N)

## 코드 예시
```c
int left = 0, right = 0;
int sum = arr[0];  // 루프 밖에서 선언, 누적 유지

while (right < N) {
    if (sum == T) break;
    else if (sum < T) {
        if (right + 1 < N) {      // 범위 초과 체크 필수!
            right++;
            sum += arr[right];
        } else break;
    }
    else {
        sum -= arr[left];
        left++;
    }
}

if (sum == T) printf("1");
else printf("0");
```

## 주의사항
- 양의 정수일 때만 성립 (음수 있으면 사용 불가)
- 연속 구간 문제는 정렬 금지 → 순서 유지해야 연속성 보장
- sum은 루프 밖에서 선언
- right++ 전에 범위 체크 필수 (right + 1 < N)
