## 한 줄 요약
> RTL 신호는 비트폭이 정확히 정해져 있어서 int 대신 stdint.h의 고정 크기 타입 사용.

## 코드 예시
```c
#include <stdint.h>

uint8_t   // 8비트  → logic [7:0]
uint16_t  // 16비트 → logic [15:0]
uint32_t  // 32비트 → logic [31:0]
uint64_t  // 64비트 → logic [63:0]
```

## 주의사항
- 일반 `int` 는 플랫폼마다 크기가 달라 RTL 모델링에 부적합
- 출력 시 `%u` (uint32_t), `%llu` (uint64_t) 형식 지정자 사용
