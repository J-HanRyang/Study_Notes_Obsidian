## 한줄 요약
> fgets로 두 파일을 동시에 한 줄씩 읽어 strcmp로 비교. golden vs result 자동 검증.

## 코드 예시
```c
while (fgets(line_g, sizeof(line_g), golden) != NULL) {
    if (fgets(line_r, sizeof(line_r), result) == NULL) {
        printf("FAIL at line %d: result 파일이 짧음\n", line_num);
        break;
    }
    if (strcmp(line_g, line_r) != 0) {
        printf("FAIL at line %d\n  golden: %s  result: %s", line_num, line_g, line_r);
    }
    line_num++;
}
if (!fail) printf("ALL PASS\n");
```

## 주의사항
- `strcmp`는 `\n`까지 비교 — 두 파일 포맷을 완전히 동일하게 맞춰야 함
- `fgets` 반환값 == NULL 체크 필수 (result가 golden보다 짧은 경우 대비)
- 두 파일 모두 `fclose` 잊지 않기
