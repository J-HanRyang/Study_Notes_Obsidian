## 한 줄 요약
> 스택은 LIFO(접시 쌓기), 큐는 FIFO(줄서기). 괄호/DFS엔 스택, BFS/순서처리엔 큐.

## 코드 예시
```c
// ── 스택 ──────────────────────────────
int stack[100];
int top = -1;

stack[++top] = val;     // push
int val = stack[top--]; // pop
int val = stack[top];   // peek
if (top == -1) { }      // isEmpty

// ── 큐 ────────────────────────────────
int queue[100];
int front = 0, rear = 0;

queue[rear++] = val;      // enqueue
int val = queue[front++]; // dequeue
if (front == rear) { }    // isEmpty
int size = rear - front;  // size
```

|         | 스택                | 큐               |
| ------- | ----------------- | --------------- |
| 원리      | LIFO              | FIFO            |
| 비유      | 접시 쌓기             | 줄서기             |
| 넣기      | push → 위에         | enqueue → 뒤에    |
| 꺼내기     | pop → 위에서         | dequeue → 앞에서   |
| isEmpty | `top == -1`       | `front == rear` |
| 활용      | 괄호 검사, 역순 출력, DFS | BFS, 순서 처리      |

## 주의사항
- 스택 비어있음 → `top == -1` (`top == 0` 아님)
- 큐 비어있음 → `front == rear`
- 괄호 검사: 단일 `()` 는 count로, 여러 종류 `()[]{}` 는 스택에 실제 값 저장
- 구조체 큐 사용 시 반드시 멤버로 접근 (`queue[i].priority`)
