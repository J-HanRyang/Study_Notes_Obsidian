## 한 줄 요약
> fopen/fprintf/fscanf/fclose 기본 흐름. 쓰기와 읽기는 따로 열어야 함.

## 코드 예시
```c
// 기본 흐름
FILE *fp = fopen("file.txt", "w");   // 쓰기 열기
FILE *fp = fopen("file.txt", "r");   // 읽기 열기
fprintf(fp, "%u\n", val);            // 파일에 쓰기
fscanf(fp, "%u", &val) == 1          // 파일에서 숫자 읽기, 성공 시 1 반환
fgets(line, sizeof(line), fp)        // 한 줄 통째로 읽기
sscanf(line, "%s %u", cmd, &val)     // 문자열 파싱
strcmp(cmd, "WRITE") == 0            // 문자열 비교, 같으면 0
fclose(fp);                          // 항상 닫기

// 문자 + 숫자 혼합 파싱 패턴
// stimulus.txt: WRITE 10 / WRITE 20 / READ
char cmd[16];
uint32_t val;

while (fscanf(fp, "%s", cmd) == 1) {
    if (strcmp(cmd, "WRITE") == 0) {
        fscanf(fp, "%u", &val);
        fifo_write(&fifo, val);
    } else if (strcmp(cmd, "READ") == 0) {
        fifo_read(&fifo);
    }
}
fclose(fp);
```

## 주의사항
- "wr" 같은 모드 없음 — 쓰기 끝나면 fclose 후 "r" 로 다시 열기
- fprintf 첫 인자는 반드시 `fp`
- fprintf 에 값 쓸 때 `&` 붙이면 안 됨
- strcmp 결과에 == 0 붙여야 같은 것 (if (strcmp(a, b)) 는 다를 때 
- fclose 빠뜨리지 않기
