## 한 줄 요약
> scanf로 배열 입력받을 때 = 붙이면 안 됨

## 코드 예시
```c
int n;
scanf("%d", &n);

int arr[n];
for (int i = 0; i < n; i++) {
    scanf("%d", &arr[i]);  // ← scanf만! 앞에 = 붙이면 안 됨
}
```

## 주의사항
- `int N = scanf("%d", &N)` ← 이렇게 하면 안 됨
- scanf는 반환값 대입 없이 그냥 호출만
