## 한 줄 요약
> srand로 시드 고정 후 rand()로 랜덤 생성. 같은 시드면 항상 같은 순서 (재현 가능).

## 코드 예시
```c
#include <stdlib.h>

srand(42);          // 시드 고정
int r = rand();     // 랜덤 숫자 생성

rand() % 256        // 0 ~ 255 (8비트)
rand() % 100        // 0 ~ 99
rand() % 2          // 0 or 1 (랜덤 enable 신호)

// ❌ 같은 r 재사용 — data가 항상 짝수
int r = rand();
if (r % 2 == 0) fprintf(fp, "WRITE %d\n", r % 100);

// ✅ 독립적으로 두 번 뽑기
int cmd  = rand() % 2;
int data = rand() % 100;
if (cmd == 0) fprintf(fp, "WRITE %d\n", data);
```

## 주의사항
- cmd용, data용 rand()를 따로 뽑아야 함 — 같은 값 재사용 시 데이터가 항상 짝수/홀수로 편향됨
- `srand()` 없이 `rand()` 호출하면 매 실행마다 같은 시퀀스 (시드 기본값 1)
- 재현 가능한 테스트가 필요하면 시드 고정, 매번 다른 결과 원하면 `srand(time(NULL))`
