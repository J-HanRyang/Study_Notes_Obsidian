## 한줄 요약
> 정렬 없이 O(n) 순회로 해결. 정렬(O(n log n))보다 효율적

## 코드 예시
```c
int min = arr[0], max = arr[0];
for (int i = 1; i < n; i++) {
    if (arr[i] < min) min = arr[i];
    if (arr[i] > max) max = arr[i];
}
```

## 주의사항
- 초기값을 arr[0]으로 잡고 i=1부터 시작
- 정렬이 필요 없을 때는 순회가 더 빠름
