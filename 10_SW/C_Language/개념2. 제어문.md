## if / else if / else

```c
if (score >= 90) {
    printf("A학점\n");
} else if (score >= 80) {
    printf("B학점\n");
} else {
    printf("F학점\n");
}
```
> Python의 `elif` → C에서는 `else if`

## switch

```c
switch (day) {
    case 1: printf("월요일"); break;
    case 3: printf("수요일"); break;
    default: printf("기타");
}
```
> `break` 없으면 다음 case까지 줄줄이 실행됨 (fall-through)

## for

```c
// for (초기값; 조건; 증감)
for (int i = 0; i < 5; i++) {
    printf("%d ", i);  // 0 1 2 3 4
}
```

## while / do-while

```c
// while — 조건 먼저 검사
int i = 0;
while (i < 5) {
    printf("%d ", i);
    i++;  // 없으면 무한루프!
}

// do-while — 최소 한 번은 실행
do {
    printf("%d\n", i);
    i++;
} while (i < 5);
```

## break & continue

```c
// break — 반복문 탈출
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    printf("%d ", i);  // 0 1 2 3 4
}

// continue — 이번 반복만 건너뜀
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;
    printf("%d ", i);  // 1 3 5 7 9
}
```

## 주의사항
- switch에서 break 빠뜨리면 fall-through 발생
- while에서 증감 빠뜨리면 무한루프
