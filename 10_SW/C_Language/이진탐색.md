## 한줄 요약
> 정렬된 배열에서 범위를 절반씩 줄여가며 탐색. O(log N)

## 순차 탐색 vs 이진 탐색
| | 순차 탐색 | 이진 탐색 |
|--|---------|---------|
| 시간복잡도 | O(N) | O(log N) |
| N=500,000 | 최대 500,000번 | 최대 19번 |
| 전제조건 | 없음 | 정렬 필요 |

## 코드 예시
```c
int binarySearch(int arr[], int n, int target) {
    int left = 0;
    int right = n - 1;

    while (left <= right) {
        int mid = (left + right) / 2;

        if (arr[mid] == target) return 1;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return 0;
}
```

## 주의사항
- 반드시 qsort 먼저 하고 이진 탐색
- left <= right (등호 빠뜨리면 안 됨)
