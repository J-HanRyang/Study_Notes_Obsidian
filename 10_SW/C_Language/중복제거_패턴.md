## 한 줄 요약
> 정렬 후 앞 원소와 다를 때만 출력 — 추가 배열 불필요

## 코드 예시
```c
qsort(arr, N, sizeof(int), compare);

printf("%d ", arr[0]);  // 첫 번째는 무조건 출력
for (int i = 1; i < N; i++) {
    if (arr[i] != arr[i-1]) printf("%d ", arr[i]);
}
```

## 주의사항
- 정렬 먼저 해야 인접 비교로 중복 제거 가능
- 추가 배열 없이 출력만으로 해결
