## 한 줄 요약
> 정렬된 배열에서 두 수의 합이 T인 쌍 찾기 — O(N)

## 이중 for문 vs 투 포인터
| | 이중 for문 | 투 포인터 |
|--|-----------|---------|
| 시간복잡도 | O(N²) | O(N) |
| 전제조건 | 없음 | 정렬 필요 |

## 코드 예시
```c
int left = 0;
int right = N - 1;  // ← 값이 아니라 인덱스!

while (left < right) {
    int sum = arr[left] + arr[right];
    if (sum == T) break;
    else if (sum > T) right--;  // 오른쪽 인덱스를 왼쪽으로
    else left++;                // 왼쪽 인덱스를 오른쪽으로
}

if (arr[left] + arr[right] == T) printf("1");
else printf("0");
```

## 주의사항
- left, right는 값이 아니라 인덱스
- 이동할 때 arr[left], arr[right]로 값에 접근
