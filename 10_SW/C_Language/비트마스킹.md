## 한 줄 요약
> 비트 AND로 wrap-around 또는 인덱스 범위 제한. max가 2의 거듭제곱일 때만 사용 가능.

## 코드 예시
```c
(cnt + 1) & ((1 << N) - 1)  // N비트 wrap-around (2의 거듭제곱일 때)
(cnt + 1) % max              // 일반적인 wrap-around
val & (DEPTH - 1)            // 배열 인덱스 wrap
ptr ^ (1 << N)               // 최상위 비트 토글 (FIFO full 판별)
```

## 주의사항
- `&` 마스킹은 max가 2의 거듭제곱(4, 8, 16...)일 때만 가능
- max=5처럼 2의 거듭제곱이 아니면 반드시 `%` 사용
- `& (max-1)` 을 습관적으로 쓰면 안 됨 — max 값 먼저 확인
