## 한 줄 요약
> N이 크면 스택 오버플로우 → malloc으로 힙에 할당

## 코드 예시
```c
// ❌ 스택 선언 — N이 100만이면 스택 오버플로우
int arr[1000000];

// ✅ 힙 할당 — 크기 제한 없음
int *arr = (int*)malloc(sizeof(int) * N);

// 다 쓰고 나면 반드시 해제
free(arr);
arr = NULL;
```

## 올바른 작업 순서
```
malloc → 입력 루프 → (정렬) → 처리 → 출력 → free → NULL
```

## 주의사항
- free는 출력까지 모든 작업이 끝난 후 맨 마지막에
- 중간에 free하면 이후 접근 시 오류
