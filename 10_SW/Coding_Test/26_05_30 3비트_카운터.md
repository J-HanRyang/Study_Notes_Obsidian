## 문제 요약
3비트 카운터 구현. 0~7 순환 후 다시 0으로 wrap-around.

## 풀이 접근
`& ((1 << N) - 1)` 마스킹으로 N비트 범위 초과 시 자동 wrap-around.

## 코드
```c
#include <stdio.h>
#include <stdint.h>

int main() {
    uint32_t cnt = 0;

    for (int i = 0; i < 10; i++) {
        cnt = (cnt + 1) & ((1 << 3) - 1);
        printf("cnt = %u\n", cnt);
    }
    return 0;
}
// 출력: 1 2 3 4 5 6 7 0 1 2
```

## 핵심 포인트
- `(1 << 3) - 1` = `0b111` = 7 → AND 마스킹으로 3비트 초과분 제거
- 2의 거듭제곱 범위일 때만 `&` 마스킹 사용 가능

## 실수 / 막혔던 것
- max가 2의 거듭제곱이 아니면 `%` 써야 함
