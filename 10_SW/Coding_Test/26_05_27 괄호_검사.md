## 문제 요약
문자열에서 괄호가 올바르게 짝지어졌는지 검사. 단일 괄호 `()` 또는 여러 종류 `()[]{}`.

## 풀이 접근
- 단일 괄호: count 증감으로 해결 — 스택 불필요
- 여러 종류: 스택에 여는 괄호를 저장, 닫는 괄호가 나올 때 top과 짝 확인

## 코드
```c
// 단일 괄호 () — count 방식
int count = 0;
for (int i = 0; i < len; i++) {
    if (str[i] == '(') count++;
    else {
        if (count == 0) { printf("0"); return 0; }
        count--;
    }
}
if (count == 0) printf("1");
else printf("0");

// 여러 종류 괄호 ()[]{}
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int main() {
    char* str = (char*)malloc(sizeof(char) * 1000);
    scanf("%s", str);

    int len = strlen(str);

    char* stack = (char*)malloc(sizeof(char) * 1000);
    int top = -1;

    for (int i = 0; i < len; i++) {
        if (str[i] == '(' || str[i] == '[' || str[i] == '{')
            stack[++top] = str[i];
        else if (str[i] == ')') {
            if (top == -1 || stack[top] != '(') { printf("0"); return 0; }
            top--;
        }
        else if (str[i] == ']') {
            if (top == -1 || stack[top] != '[') { printf("0"); return 0; }
            top--;
        }
        else if (str[i] == '}') {
            if (top == -1 || stack[top] != '{') { printf("0"); return 0; }
            top--;
        }
    }

    if (top == -1) printf("1");
    else printf("0");

    free(str);
    free(stack);
    str = NULL;
    stack = NULL;

    return 0;
}
```

## 핵심 포인트
- `)` 가 나왔을 때 **직전에 열린 괄호가 뭔지** 알아야 하므로 스택에 실제 값 저장
- LIFO 원리상 top = 가장 최근에 열린 괄호
- 단일 괄호는 count, 여러 종류는 반드시 스택

## 실수 / 막혔던 것
- 닫는 괄호 나왔을 때 `top == -1` 체크를 먼저 해야 함 (순서 중요)
- 끝까지 순회 후 `top == -1` 이어야 유효 — 마지막 체크 빠뜨리지 않기
