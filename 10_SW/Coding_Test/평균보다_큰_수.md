## 문제 요약
N개의 정수에서 평균보다 큰 수의 개수 출력

## 풀이 접근
합계 구하고 평균 계산 → 평균보다 큰 수 카운트
정수 나눗셈 함정 주의 → (double) 형변환 필요

## 코드
```c
#include <stdio.h>

int main() {
    int N;
    scanf("%d", &N);

    int array[N];
    int sum = 0;
    for (int i = 0; i < N; i++) {
        scanf("%d", &array[i]);
        sum += array[i];
    }

    double avg = (double)sum / N;
    int count = 0;
    for (int j = 0; j < N; j++) {
        if (array[j] > avg) count++;
    }

    printf("%d", count);
    return 0;
}
```

## 핵심 포인트
- `(double)sum / N` — 정수 나눗셈 함정 주의
- `double(sum)` 은 C에서 안 됨 → `(double)sum` 으로

## 실수 / 막혔던 것
- 형변환 문법 실수: `double(sum)` → `(double)sum`
