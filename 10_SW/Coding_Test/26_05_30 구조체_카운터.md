## 문제 요약
구조체로 카운터 상태 관리. count와 max를 묶어 tick 함수로 순환 증가.

## 풀이 접근
구조체 포인터로 넘겨 원본 상태 수정. `% max` 로 wrap-around.

## 코드
```c
typedef struct {
    uint32_t count;
    uint32_t max;
} counter_t;

void counter_tick(counter_t *c) {
    c->count = (c->count + 1) % c->max;
}
// count=0, max=5 → 8번 tick: 1 2 3 4 0 1 2 3
```

## 핵심 포인트
- 구조체를 값으로 넘기면 복사본만 바뀜 → 반드시 포인터로 넘기기
- max=5는 2의 거듭제곱이 아니므로 `%` 사용

## 실수 / 막혔던 것
- `c->max` 대신 `max` 로 쓰면 컴파일 에러 (스코프 밖)
- `void f(counter_t c)` 로 넘기면 원본 안 바뀜
