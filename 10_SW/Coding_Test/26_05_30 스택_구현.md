## 문제 요약
구조체로 스택 구현. push/pop 함수에서 size로 상태 관리.

## 풀이 접근
구조체 포인터로 원본 수정. push는 size < max 체크, pop은 size == 0 체크.

## 코드
```c
typedef struct {
    uint32_t data[4];
    uint32_t size;
} stack_t;

void stack_push(stack_t *s, uint32_t val) {
    if (s->size < 4) {
        s->data[s->size] = val;
        s->size += 1;
        printf("PUSH: val=%u, size=%u\n", val, s->size);
    }
}

void stack_pop(stack_t *s) {
    if (s->size == 0) {
        printf("FAIL (empty)\n");
    } else {
        printf("POP: val=%u, size=%u\n", s->data[s->size - 1], s->size - 1);
        s->size -= 1;
    }
}
```

## 핵심 포인트
- `stack_t s = {0}` 으로 전체 0 초기화
- push/pop 모두 반드시 포인터로 넘겨야 원본 수정 가능
- pop 시 `s->data[s->size - 1]` 로 마지막 원소 접근

## 실수 / 막혔던 것
- `void stack_push(stack_t s, ...)` 로 값으로 넘기면 원본 안 바뀜
- size 체크 없이 push/pop 하면 범위 초과
