## 문제 요약
N개(홀수)의 정수에서 중앙값 출력

## 풀이 접근
정렬 후 arr[N/2]

## 코드
```c
#include <stdio.h>
#include <stdlib.h>

int compare(const void* a, const void* b) {
    return *(int*)a - *(int*)b;
}

int main() {
    int N;
    scanf("%d", &N);

    int *arr = (int*)malloc(sizeof(int) * N);
    for (int i = 0; i < N; i++) {
        scanf("%d", &arr[i]);
    }

    qsort(arr, N, sizeof(int), compare);

    int mid = N / 2;
    printf("%d", arr[mid]);

    free(arr);
    arr = NULL;

    return 0;
}
```

## 핵심 포인트
- 정렬 후 arr[N/2] → N이 홀수일 때 중앙값
