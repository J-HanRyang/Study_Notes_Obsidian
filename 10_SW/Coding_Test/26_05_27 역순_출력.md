## 문제 요약
N개의 정수를 입력받아 역순으로 출력

## 풀이 접근
스택에 push로 다 넣고 pop으로 꺼내면 자동으로 역순

## 코드
```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    int N;
    scanf("%d", &N);

    int *stack = (int*)malloc(sizeof(int) * N);
    int top = -1;

    for (int i = 0; i < N; i++) {
        int val;
        scanf("%d", &val);
        stack[++top] = val;
    }

    for (int i = 0; i < N; i++) {
        printf("%d ", stack[top--]);
    }

    free(stack);
    stack = NULL;

    return 0;
}
```

## 핵심 포인트
- push로 다 넣고 pop으로 꺼내면 자동으로 역순
- 동적 할당 사용 시 `free`는 출력 끝난 뒤

## 실수 / 막혔던 것
- `free` 위치를 출력 전에 두면 안 됨
