## 문제 요약
N개의 정수에서 최솟값과 최댓값 출력

## 풀이 접근
정렬보다 순회가 더 효율적 → O(n)으로 해결

## 코드
```c
#include <stdio.h>

int main() {
    int N;
    scanf("%d", &N);

    int array[N];
    for (int i = 0; i < N; i++) {
        scanf("%d", &array[i]);
    }

    int min = array[0], max = array[0];
    for (int i = 1; i < N; i++) {
        if (array[i] < min) min = array[i];
        if (array[i] > max) max = array[i];
    }

    printf("%d %d", min, max);
    return 0;
}
```

## 핵심 포인트
- 정렬(O(n log n)) 쓸 필요 없이 순회(O(n))로 해결
